# 科技风演示文稿生成器

基于 AI 的科技风格 PPT 生成器，输出单文件 HTML 格式的演示文稿。

## 功能特点

- **极简科技风格**：深色渐变背景 + 动态光效
- **9:16 竖屏比例**：适合移动端和现代演示场景
- **多种幻灯片类型**：
  - 封面页（大标题 + 副标题 + 日期）
  - 标题页（章节序号 + 章节标题 + 说明）
  - 引用页（引用内容 + 出处）
  - 步骤页（步骤列表 + 详细说明）
  - 结束页（感谢语 + 联系方式）
- **键盘导航**：支持左右键切换
- **触摸滑动**：支持移动端滑动切换
- **单文件交付**：所有资源内置，双击即可使用

## 使用方法

1. 双击打开 `ppt-generator.html` 文件
2. 在输入框中输入演示文稿内容
3. 点击"生成演示文稿"按钮
4. 使用键盘左右键或底部导航栏切换幻灯片

## 输入格式

```
封面标题：你的演示标题
副标题：演讲者或副标题信息
日期：2026 年 3 月

第一章：章节标题
章节简要说明或内容

第二章：另一个章节
更多内容说明

第三章：技术要点
要点一
要点说明
要点二
要点说明

引用：引用内容 - 作者

结束页信息：感谢观看
联系方式：contact@example.com
关注公众号获取更多信息
```

## 技术栈

- **HTML5**：基础结构
- **TailwindCSS**（CDN）：样式框架
- **自定义 CSS**：动画和科技感样式
- **JavaScript**：交互逻辑

## 浏览器支持
